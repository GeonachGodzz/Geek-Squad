Introduction

This case study investigates how different hardware components such as CPUs, GPUs, and cooling systems affect gaming performance. It focuses on frame rate, temperature, and system stability during gameplay. The goal is to determine if better hardware leads to significantly better gaming experiences.

Now what is it that we are trying to find out? These are the questions we hope to answer:

  1. Does brand matter when buying hardware?

  2. How does temperature affect game performance?

  3. Can a CPU run heavy games without a GPU?

  4. Do games run better on Windows or Linux?
